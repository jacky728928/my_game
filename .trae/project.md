# 生存射击 · v0.3

手机俯视角生存 | Canvas 原生 JS | 摇杆操控

## 文件地图 & 改哪里
| 文件 | 内容 | 改什么 |
|------|------|--------|
| `index.html` | 入口/加载顺序 | 新增JS引用 |
| `css/style.css` | 样式 | 摇杆/布局 |
| `js/constants.js` | 所有常量 + ABILITY_POOL + 暴击参数 | 数值/敌人预设/颜色/新能力 |
| `js/player.js` | 玩家(HP/移动/攻速加成/暴击率/暴击倍率/等级/能力应用) | 玩家逻辑/新能力 |
| `js/enemy.js` | 敌人(追踪/碰撞/6级) | 敌人AI |
| `js/bullet.js` | 子弹(飞行/碰撞/isCrit标记) | 子弹逻辑 |
| `js/xp_orb.js` | 经验球(浮动/飞向吸收) | 经验球逻辑 |
| `js/input.js` | 虚拟摇杆 | 操控 |
| `js/minimap.js` | 左上小地图 | 小地图 |
| `js/renderer.js` | 主渲染(背景/实体/血条/伤害数字/能力状态) | 画面/UI |
| `js/main.js` | 主循环(生成/碰撞/升级暂停/三选一能力选择) | 游戏循环 |

## 核心参数速查
```
世界 2400×2400 | 网格80px
玩家 radius=18 speed=180 maxHp=100
手枪 range=250 dmg=5 interval=1.0s | bullet speed=400 radius=4
敌人 Lv1-6: HP=10~60, size=22~42, speed 50~155, 每级不同色
生成间隔 2.5→0.6 | 难度随时间解锁更高等级
经验 pickup=70px | 升级=等级×20 | 奖励+5maxHP | 球飞向角色420px/s
碰撞=敌人撞玩家→玩家扣敌人HP | 子弹命中→扣伤害值
暴击基础倍率 1.5 | 攻速加成后间隔=基础间隔 × (100 / (100 + 攻速加成))
能力池（7选3随机）：hp_max | attack_speed | crit_chance | damage(+2) | view_range(+5%) | attack_range(+5px) | pickup_range(+5px)
```
