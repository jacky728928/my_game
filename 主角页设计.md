# 主页 · 主角标签页设计文档（v3 —— 精简版）

## 一、页面定位

**主角页 = 角色选择页**
- 玩家初始拥有角色：**战术干员 Echo-01**（`pic/初始角色.png`）
- 在「冒险」页的 `↻ 切换立绘` 按钮旁边，新增一个 **选择角色** 按钮，点击即跳转到本页
- 用户在此选择**本局出战角色**
- 每个角色有独特外观、属性加成、专属主动技能
- 副武器在**游戏内**通过升级选择随机获取，与本页无关

## 六、交互流程

### 6.1 入口：从冒险页进入
- 冒险页（`data-page="adventure"`）在「开始游戏」按钮上方有一个 **选择角色** 按钮
- 点击后：切换到底部导航的「主角 ◉」标签（即 `data-page="hero"`）

### 6.2 切换角色
1. 点击中部角色卡片
2. 选中态切换：金色描边 `#d4af37` + 外投影 `0 0 20px rgba(212,175,55,0.3)`
3. 底部详情区动画刷新为新角色（淡入淡出）
4. 写入 `localStorage`（key：`hero_selected_id`）

### 6.3 确认返回
- 点击底部「确认选择 / 返回冒险」按钮
- 自动切换回「冒险」页，并使用新选中的角色
