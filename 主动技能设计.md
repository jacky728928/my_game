# 主动技能设计文档

> 版本：v0.1 | 状态：**已实装**

---

## 一、主动技能总览

主动技能是开局三选一获得的特殊能力，每个玩家同一时间只能装备 **1 个**主动技能。与副武器专属词条不同，主动技能本身不通过升级获得进阶词条（后续可扩展），释放方式为：

| 平台 | 操作 |
|------|------|
| 手机 | 按住右下角圆形图标瞄准 → 松开释放 |
| 电脑 | 按住 E 键瞄准 → 松开释放 |

> 闪烁突袭显示**落点预览**（虚线圆 + 连线 + 准星）；其他技能按下即释放。

## 二、闪烁突袭 — ✦ 瞬移轰炸 ⚡

> 紫色主题 | 爆发型技能

| 属性 | 值 |
|------|----|
| 技能 ID | `blink` |
| 图标 | `✦` |
| 主题色 | 紫色 `#9b59b6` |
| 瞬移距离 | 300 px |
| AOE 半径 | **150 px**（`aoeRadius * 3`） |
| 伤害 | 基础攻击 × **200%** |
| 冷却时间 | **10 秒** |
| 霸体时间 | 落地后 **0.2 秒**（防重叠扣血） |
| 状态 | **已实装** |

### 触发流程

1. **按住** E / 图标 → 进入瞄准充能状态，画面显示落点预览（虚线 AOE 圆 + 准星）
2. **松开** → 玩家瞬间移动至落点
3. 落点位置触发 AOE 伤害结算（半径内敌人扣血 + 暴击数字）
4. 起地位置喷出紫色光圈（玩家原来位置）
5. 落地位置喷出多层爆炸视觉 + 粒子 + 屏幕震动 + 镜头闪光
6. 落地后 0.2 秒霸体保护

### 落地特效列表

| 效果 | 类型 | 描述 |
|------|------|------|
| 起地光圈 ×2 | `ring_main` / `ring_outer` | 玩家原位置的紫色冲击波 |
| 白色强光闪光 | `flash` | 落地瞬间全屏白光 |
| 主紫环 | `ring_main` | 粗线紫色冲击波 |
| 白色内光环 | `ring_white` | 中心亮环 |
| 外圈扩散环 | `ring_outer` | 大范围淡紫环 |
| 内层脉冲 | `ring_inner` | 快速收缩 |
| 烧焦印记 | `scorch` | 地面残留 1.4 秒 |
| 粒子 ×54 | 散射粒子 | 三层粒子分布 |
| 屏幕震动 | `screenShake` | 强度 14，时长 0.35s |
| 镜头闪光 | `cameraFlash` | 强度 0.25，时长 0.18s |

## 三、冲刺翻滚 — ➤ 闪避位移 🌀

> 蓝色主题 | 机动型技能

| 属性 | 值 |
|------|----|
| 技能 ID | `dash` |
| 图标 | `➤` |
| 主题色 | 蓝色 `#3498db` |
| 位移距离 | 200 px（分 6 步位移） |
| 霸体时间 | **0.3 秒**（全程免伤） |
| 冷却时间 | **3 秒** |
| 状态 | **已实装** |

### 触发流程

1. **按住 / 松开**（无充能）→ 立即向摇杆方向快速位移
2. 位移分 **8** 步执行，每步留下蓝色粒子
3. 霸体 **全程覆盖** 0.3 秒，敌人无法造成碰撞伤害
4. 冷却 3 秒

### 位移轨迹粒子

每步位移时在当前位置生成 1 颗蓝色粒子（`life: 0.4s`），累积形成流畅的冲刺拖尾效果。

## 四、疾跑 — » 移速爆发 🏃

> 绿色主题 | 增益型技能

| 属性 | 值 |
|------|----|
| 技能 ID | `haste` |
| 图标 | `»` |
| 主题色 | 绿色 `#2ecc71` |
| 移速倍率 | **160%**（基础速度 × 1.6） |
| 持续时间 | **5 秒** |
| 冷却时间 | **5 秒** |
| 状态 | **已实装** |

### 触发流程

1. **按住 / 松开**（无充能）→ 立即获得移速加成
2. 移速提升至 `基础速度 × 1.6`，持续 5 秒
3. 持续期间：玩家速度实时更新，buff 倒计时
4. buff 结束后速度恢复正常
5. 冷却 5 秒

### 速度计算

```
实际速度 = 基础速度 × speedMult  （疾跑期间）
基础速度 = PLAYER_SPEED = 180 px/s
疾跑速度 = 180 × 1.6 = 288 px/s
```

## 五、代码映射

| 文件 | 内容 |
|------|------|
| `js/constants.js` | `ACTIVE_SKILL_POOL`（技能静态数据）+ `ACTIVE_SKILL_LIST`（列表） |
| `js/player.js` | `Player.activeSkillId / activeSkillCooldown / activeSkillBuff / _invulnTimeLeft` + `equipActiveSkill / releaseActiveSkill / isInvulnerable / startInvuln` |
| `js/active_skill.js` | `releaseActiveSkillNow()`（释放逻辑）+ `getActiveSkillPreview()`（落点预览，含墙体回退修正）+ `onActiveSkillPress/Release()`（输入处理）+ `ensureActiveSkillIcon()`（图标 DOM）+ `_updateSkillIconVisual()`（冷却环/呼吸发光） |
| `js/renderer.js` | 落点预览绘制（调用 `getActiveSkillPreview()`） |

### 核心函数说明

| 函数 | 位置 | 作用 |
|------|------|------|
| `_getActiveSkillDirection()` | `js/active_skill.js` | 获取摇杆方向或朝向（归一化），若不动则使用 `player.angle` |
| `getActiveSkillPreview()` | `js/active_skill.js` | 返回落点坐标（含墙体回退：落点进入墙体时向玩家方向回退到合法位置），供渲染器绘制预览 |
| `_canReleaseActiveSkill()` | `js/active_skill.js` | 释放前置检查（存活/非暂停/冷却就绪） |
| `releaseActiveSkillNow()` | `js/active_skill.js` | 执行技能效果（blink 瞬移+AOE+粒子/dash 位移+墙体回退/haste 移速 buff），并把冷却写入 `player.activeSkillCooldown` |
| `onActiveSkillPress()` | `js/active_skill.js` | 按下事件 → blink 进入充能瞄准，其他立即释放 |
| `onActiveSkillRelease()` | `js/active_skill.js` | 松开事件 → 若为充能状态则释放 |
| `ensureActiveSkillIcon()` | `js/active_skill.js` | 创建/更新右下角圆形图标 DOM + pointer 事件 |
| `_updateSkillIconVisual()` | `js/active_skill.js` | 刷新图标视觉（就绪/冷却环/呼吸发光 + 提示文字） |

### 全局状态

```javascript
let _activeSkillCharging = false;  // 是否处于"按住瞄准"状态（仅 blink 有效）
```

## 六、UI 表现

### 技能图标（右下角圆形按钮）

| 状态 | 视觉 |
|------|------|
| 就绪 | 技能图标 + 渐变背景 + "按 E / 点击" |
| 冷却中 | conic-gradient 冷却环 + 剩余秒数 |
| 充能瞄准中（仅 blink） | 呼吸发光 + "瞄准中·松开释放" |

### 落点预览（仅 blink 充能中）

- 玩家到落点的**虚线连线**（流动动画）
- 落点处 **AOE 虚线圆**（呼吸脉冲）
- 落点处**浅色渐变填充**（增强范围感）
- 落点中心**白色十字准星**
- 落点中心**彩色小圆点**（带发光）
