# 角色设计 - 初始角色

## 角色设定

**名称**：战术干员 / 代号 "Echo-01"

**外观特征**
- 黑色战术制服、头盔 + 战术护目镜
- Q 版像素风，紧凑型设计
- 主武器：**等离子冲锋枪**（蓝紫发光，能量型武器）

**整体风格**：精英战术干员，装备先进能量武器

---

## 主武器

### 等离子冲锋枪 (Plasma Carbine)

| 属性 | 初始值 | 备注 |
|------|--------|------|
| 攻击间隔 | 1.0 秒 | 三连发总冷却 |
| 三连发间隔 | 0.12 秒 | 发与发之间 |
| 每发伤害 | 基础攻击 × 40% | 单颗伤害较低，靠连发输出 |
| 总轮伤害 | 基础攻击 × 120% | 三连发加总略高于单发 |
| 攻击范围 | 260 px | 中远距离 |
| 弹速 | 420 | 中等弹速 |
| 子弹颜色 | `#8a5cf0`（紫蓝） | 带内芯 `#ffffff` |
| 子弹外形 | 椭圆能量弹（带发光拖尾） |
| 暴击颜色 | `#ff3366` | 高亮红 |

**视觉特点**
- 三连发时，第一发枪管有明显能量闪光，后续两发快速跟随
- 子弹为椭圆形，带蓝紫色发光拖尾（类似能量弹）
- 命中时产生蓝紫能量爆炸粒子
- 连发间隔期间枪口有持续的弱光点

---

## 主动技能

### 脉冲冲刺 (Pulse Dash)

| 属性 | 值 |
|------|-----|
| 冷却时间 | 6 秒 |
| 效果描述 | 向摇杆方向快速冲刺 180 px，冲刺期间霸体；冲刺路径上的敌人受到基础伤害 × 2 的伤害并被击退 |
| 视觉 | 蓝紫色能量尾迹 + 粒子爆炸 |
| 图标色 | `#9b59b6` |

---

## 被动技能

### 战术充能 (Tactical Charge)

| 属性 | 值 |
|------|-----|
| 触发条件 | 每完成一轮三连发记 1 次，累计 5 轮后自动充能一层 |
| 充能效果 | 下一轮三连发的首发子弹伤害 +100%（单颗 × 1.8），并使敌人短暂硬直 0.3 秒 |
| 最大层数 | 3 层 |
| 视觉 | 角色身上有能量环圈，层数越高环越亮；充能时武器枪口紫光变亮 |

### 战场反应 (Field Reflex)

| 属性 | 值 |
|------|-----|
| 效果描述 | 生命值低于 30% 时，移动速度 +20%，攻速 +15% |
| 视觉 | 角色光圈变为红色警告色 |

## 视觉表现参考

```
玩家渲染顺序：
  1. 能量环圈（被动充能层数对应大小/亮度）
  2. 外圈（蓝色发光）
  3. 本体（黑色圆形/战术轮廓）
  4. 朝向指示（蓝紫色能量枪管）
  5. 武器发射粒子（攻击瞬间）

子弹渲染：
  核心：白色能量核
  外圈：紫蓝发光椭圆
  拖尾：渐淡紫蓝光带
```
