# 游戏设计文档 — 生存射击

## 概念

手机端俯视角生存射击。玩家固定在屏幕中央，敌人从四面八方涌来，玩家用武器消灭敌人以求存活。

## 核心循环

```
移动避敌 → 自动攻击 → 击杀 → 生存时间累积 → 敌人增强 → 死亡/重新开始
```

## 技术栈

- 纯前端 HTML5 Canvas
- 原生 JS，零依赖
- 面向手机触屏设计，同时兼容鼠标

## 世界参数

| 参数 | 值 |
|------|-----|
| 世界尺寸 | 2400 × 2400 px |
| 网格 | 80px 间距 |
| 摄像机 | 锁定跟随玩家 |

## 角色参数

| 参数 | 值 |
|------|-----|
| 初始 HP | 100 |
| 移动速度 | 180 px/s |
| 外形 | 蓝色圆形 r=18px |

## 武器参数

| 参数 | 值 |
|------|-----|
| 武器名 | 小手枪 |
| 攻击间隔 | 1.0 s |
| 伤害 | 5 |
| 范围 | 250 px |
| 子弹速度 | 400 px/s |
| 瞄准 | 自动锁最近敌人 |

## 敌人参数

| 等级 | HP | 大小 | 速度区间 | 颜色 |
|------|-----|------|----------|------|
| Lv1 | 10 | 22px | 100-155 | 红 |
| Lv2 | 20 | 26px | 90-145 | 橙 |
| Lv3 | 30 | 30px | 80-135 | 黄 |
| Lv4 | 40 | 34px | 70-125 | 绿 |
| Lv5 | 50 | 38px | 60-115 | 蓝 |
| Lv6 | 60 | 42px | 50-105 | 紫 |

- 生成位置：视口边缘外侧 60px
- AI：直线追踪玩家
- 碰撞：敌人接触玩家 → 敌人死亡，玩家扣等于敌人当前 HP 的伤害
- 难度递进：游戏时长越长，可出现的最高等级越高
