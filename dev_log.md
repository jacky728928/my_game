## 项目概要

手机端俯视角生存射击，Canvas 渲染。主角固定在屏幕中央，虚拟摇杆移动，自动瞄准攻击。

```
├── input.js           # 虚拟摇杆 (触屏+鼠标)
├── minimap.js         # 左上角小地图 (含墙体绘制)
├── renderer.js        # 主画布渲染 (背景/墙体/实体/血条/UI/伤害数字/准星)
├── effects.js         # 特效系统 (爆炸/粒子/目标标记/轨道炮蓄力/发射/爆炸)
├── combat.js          # 战斗系统 (攻击/副武器更新含副武器专属词条/AOE伤害/墙体碰撞检测)
├── active_skill.js    # 主动技能系统 (3个技能释放/蓄力瞄准/图标更新/墙体碰撞)
├── ui.js              # UI界面 (升级选择/模式选择/副武器选择/主动技能选择/丢弃UI/副武器专属词条混入)
├── dev_panel.js       # 开发者面板 (属性加成/武器开关/经验倍率)
└── game_core.js       # 游戏循环/状态机/敌人生成(避开墙体)/游戏重启
```
