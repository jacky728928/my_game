# 开发日志 — 生存射击

> 最后更新：2026-06-13

## 项目概要

手机端俯视角生存射击，Canvas 渲染。主角固定在屏幕中央，虚拟摇杆移动，自动瞄准攻击。

---

## 版本记录

### v0.1 — 2026-06-13 · 基础骨架
- [x] 8 文件模块化架构（index.html + 7 JS + 1 CSS）
- [x] 主角：圆形，固定屏幕中央，100HP，受击闪光
- [x] 转轮操控：140px 虚拟摇杆，支持触屏 + 鼠标
- [x] 敌人：方形，6 级预设（Lv1-6，HP=等级×10）
  - Lv1: 10HP, 22px, 红, 100-155移速
  - Lv2: 20HP, 26px, 橙, 90-145
  - Lv3: 30HP, 30px, 黄, 80-135
  - Lv4: 40HP, 34px, 绿, 70-125
  - Lv5: 50HP, 38px, 蓝, 60-115
  - Lv6: 60HP, 42px, 紫, 50-105
- [x] 敌人从视口边缘生成，追踪玩家，碰撞后敌人死亡、主角扣等同血量
- [x] 武器：小手枪，1s 间隔，5 伤害，250px 范围，自动瞄准最近敌人
- [x] 子弹：黄色圆点，400px/s，带光晕
- [x] 血量条：画在主角头顶（Canvas 内），颜色按比例变化
- [x] 小地图：左上角 120×120，半透明，显示世界全貌
- [x] 世界：2400×2400，深蓝网格背景
- [x] 击杀计数 + 死亡/重开机制
- [x] 敌人难度递进：随时间逐步解锁更高等级

## 架构速查

```
js/
├── constants.js   # 所有数值常量、6级敌人预设、颜色
├── player.js      # 玩家类 (HP/移动/攻击冷却/受伤)
├── enemy.js       # 敌人类 (等级/追踪/碰撞)
├── bullet.js      # 子弹类 (飞行/寿命/碰撞)
├── input.js       # 虚拟摇杆 (触屏+鼠标)
├── minimap.js     # 左上角小地图
├── renderer.js    # 主画布渲染 (背景/实体/血条/UI)
└── main.js        # 游戏循环/生成/碰撞/状态机
```
