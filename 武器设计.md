# 武器设计文档

> 版本：v0.4 | 状态：**已实装**

---

## 一、武器系统总览

### 武器分类
- **主武器**：1 件（自动攻击，不可替换）
- **副武器**：最多 3 件（升级时选择装备）

### 副武器获取时机
- 第 5 级
- 第 10、20、30… 级（10n 级）

### 槽位管理
- 最多同时装备 **3 个**副武器
- 超过上限时需手动丢弃已装备的武器才能装备新的

## 二、主武器 — 手枪（Pistol）

| 属性 | 值 |
|------|----|
| 基础攻击间隔 | 1.0 秒 |
| 基础伤害 | 5 点 |
| 攻击范围半径 | 250 px |
| 子弹速度 | 400 px/s |
| 子弹半径 | 4 px |
| 攻击方式 | 自动瞄准最近敌人 → 发射子弹 |
| 伤害公式 | `effectiveDamage × 攻速加成 × 暴击倍率` |

**动态属性**：
- 攻速加成来自能力池选择（降低攻击间隔）
- 暴击率 + 暴击倍率来自能力池选择
- 基础攻击伤害来自能力池选择（`damageBonus`）

**墙体交互**：
- 矮墙：子弹穿过（不阻挡）
- 中墙 / 高墙：子弹被阻挡（命中墙体后子弹消失，生成小型撞击粒子）

## 三、副武器 — 榴弹炮（Grenade Launcher）🔶

| 属性 | 值 |
|------|----|
| ID | `grenade_launcher` |
| 类型 | `aoe_direct`（直接范围伤害） |
| 冷却时间 | 5 秒 |
| 伤害倍率 | 基础攻击 × 800%（×8.0） |
| AOE 半径 | 100 px |
| 特效颜色 | 橙色 `#e67e22` |
| 专属词条 | **火力倾泻**（`gl_firepower`）：每级额外发射 1 颗榴弹，各自锁定不同敌人 |
| 状态 | **已实装** |

**攻击流程**：
1. 冷却结束后，搜索最近敌人（范围无限），若装备专属词条则按等级搜索多名
2. 每个目标位置显示目标锁定指示器（橙色圆圈）
3. 榴弹从玩家位置向目标飞行（动态飞行时间：`max(0.5, min(2.0, 距离/300))` 秒）
4. 飞行路径中持续喷射橙色火花/烟雾粒子
5. 到达目标位置 → AOE 范围伤害 + 多层冲击波爆炸特效 + 白色强光闪光
6. 墙体交互：矮墙/中墙不阻挡飞行；命中高墙则提前爆炸（触发相同 AOE 伤害）

**伤害公式**：`player.effectiveDamage × 8.0`
- 基础状态（攻击=5）：**40 伤害 / 每颗**
- +2 伤害加成（攻击=7）：**56 伤害 / 每颗**
- 若拥有 2 级火力倾泻：同时发射 3 颗 → 总伤害 ×3

## 四、副武器 — 手雷（Grenade）🔴

| 属性 | 值 |
|------|----|
| ID | `grenade` |
| 类型 | `aoe_delayed`（延时范围伤害） |
| 冷却时间 | 10 秒 |
| 伤害倍率 | 基础攻击 × 1500%（×15.0） |
| AOE 半径 | 100 px |
| 引信时间 | 2 秒（落地后延时爆炸） |
| 飞行时间 | `max(0.5, min(1.0, 距离/300))` 秒（随距离动态） |
| 特效颜色 | 红色 `#e74c3c` |
| 专属词条 | **集束手雷**（`gr_cluster`）：每级伤害 +50%（`damageMult × (1 + 0.5 × 等级)`） |
| 状态 | **已实装** |

**攻击流程**：
1. 自动搜索玩家附近敌人（300 px 内）
2. 手雷从玩家向目标位置飞行，带橙色火花尾迹
3. 落地后进入引信阶段：
   - 持续喷射火花粒子（引信进度越长，粒子越密集）
   - 落地位置周围生成 8 颗碎片粒子
4. 墙体交互：矮墙/中墙不阻挡飞行；命中高墙则提前落地并开始引信
5. 2 秒引信结束 → AOE 范围伤害 + 多层冲击波爆炸

**伤害公式**：`player.effectiveDamage × 15.0 × (1 + 0.5 × gr_cluster_等级)`
- 基础状态（攻击=5，0 级词条）：**75 伤害**
- +2 伤害加成（攻击=7，0 级词条）：**105 伤害**
- 基础状态 + 2 级集束手雷：`5 × 15 × 2.0 =` **150 伤害**

## 五、副武器 — 医疗包（Medkit）💚

| 属性 | 值 |
|------|----|
| ID | `medkit` |
| 类型 | `heal`（回复生命） |
| 冷却时间 | 8 秒 |
| 回复倍率 | 基础攻击 × 2000%（×20.0） |
| 特效颜色 | 绿色 `#2ecc71` |
| 专属词条 | **急救强化**（`mk_healboost`）：每级回复量 +50%（`damageMult × (1 + 0.5 × 等级)`） |
| 状态 | **已实装** |

**攻击流程**：
1. 自动触发（无需目标）
2. 按当前等级计算治疗值，恢复玩家生命值，不超过最大 HP
3. 玩家头顶弹出绿色 "+数值" 治疗数字

**回复公式**：`player.effectiveDamage × 20.0 × (1 + 0.5 × mk_healboost_等级)`
- 基础状态（攻击=5，0 级词条）：**+100 HP**
- +2 伤害加成（攻击=7，0 级词条）：**+140 HP**
- 基础状态 + 2 级急救强化：`5 × 20 × 2.0 =` **+200 HP**

## 六、副武器 — 天基屠龙炮（Orbital Cannon）🛰️

| 属性 | 值 |
|------|----|
| ID | `orbital_cannon` |
| 类型 | `orbital`（空对地轰炸） |
| 冷却时间 | 60 秒（每次击杀敌人额外 -2 秒冷却） |
| 伤害倍率 | 基础攻击 × 2000%（×20.0） |
| AOE 半径 | 300 px |
| 特效颜色 | 青色 `#00d4ff` |
| 墙体交互 | **无视所有墙体**（轨道光束从天空直落） |
| 状态 | **已实装** |

**攻击流程**：
1. 冷却结束后，计算所有存活敌人的位置平均值作为轰炸中心
2. 显示目标锁定指示器（青色大圆，300 px 范围）
3. 蓄力阶段（约 1 秒）：
   - 能量汇聚粒子向中心收缩
   - 多层收缩准星逐渐聚焦
   - 屏幕警告 + 进度条提示
4. 发射阶段：
   - 轨道光束从上方直击目标位置
   - 屏幕震动 + 镜头闪光
5. 爆炸阶段：
   - 300 px AOE 范围伤害
   - 多层冲击波 + 大量碎片粒子 + 闪电链 + 烧焦印记残留
6. 击杀反馈：每击杀 1 个敌人，冷却计时器额外 -2 秒

**伤害公式**：`player.effectiveDamage × 20.0`
- 基础状态（攻击=5）：**100 伤害**
- +2 伤害加成（攻击=7）：**140 伤害**

## 七、通用机制

### 伤害挂钩公式
所有副武器的数值都与玩家当前基础攻击力挂钩：

```
实际数值 = player.effectiveDamage × weapon.damageMult
```

其中 `effectiveDamage = PISTOL_DAMAGE(5) + damageBonus(来自能力升级)`

### 副武器专属词条与升级
- 词条抽取：`js/ui.js → pickRandomChoices()`（从通用能力池 + 已装备副武器对应的专属词条池混合抽取）
- 词条应用：`js/player.js → applyAbility()`（副武器专属词条走 default 分支，累加 `secondaryAbilityLevels`）
- 生效位置：`js/combat.js → updateSecondaryWeapons()`（各武器分支读取词条等级）

### 开发控制
开发者面板（按 **D 键**切换）可独立控制：
- 主武器攻击开启/关闭
- 1 号副武器开启/关闭
- 2 号副武器开启/关闭
- 3 号副武器开启/关闭
- 经验倍率切换（1x / 5x / 10x）

### 已解决的 Bug
- [x] 副武器初始冷却值错误 → 改为 `cooldown=0`（装备后立即生效）
- [x] Enemy 类缺少 `radius` 属性 → AOE 判定 NaN → 永远扣血失败
