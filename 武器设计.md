# 武器设计文档

> 版本：v0.3 | 状态：**已实装**

---

## 一、武器系统总览

### 武器分类
- **主武器**：1 件（自动攻击，不可替换）
- **副武器**：最多 3 件（升级时选择装备）

### 副武器获取时机
- 第 5 级
- 第 10、20、30… 级（10n 级）

### 槽位管理
- 最多同时装备 **3 个**副武器
- 超过上限时需手动丢弃已装备的武器才能装备新的

## 二、主武器 — 手枪（Pistol）

| 属性 | 值 |
|------|----|
| 基础攻击间隔 | 1.0 秒 |
| 基础伤害 | 5 点 |
| 攻击范围半径 | 250 px |
| 子弹速度 | 400 px/s |
| 子弹半径 | 4 px |
| 攻击方式 | 自动瞄准最近敌人 → 发射子弹 |
| 伤害公式 | `effectiveDamage × 攻速加成 × 暴击倍率` |

**动态属性**：
- 攻速加成来自能力池选择（降低攻击间隔）
- 暴击率 + 暴击倍率来自能力池选择
- 基础攻击伤害来自能力池选择（`damageBonus`）

## 三、副武器 — 榴弹炮（Grenade Launcher）🔶

| 属性 | 值 |
|------|----|
| ID | `grenade_launcher` |
| 类型 | `aoe_direct`（直接范围伤害） |
| 冷却时间 | 5 秒 |
| 伤害倍率 | 基础攻击 × 800%（×8.0） |
| AOE 半径 | 100 px |
| 特效颜色 | 橙色 `#e67e22` |
| 状态 | **已实装** |

**攻击流程**：
1. 自动搜索最近敌人（范围无限）
2. 敌人位置显示目标锁定指示器（橙色圆圈，0.45 秒）
3. 瞬间触发 AOE 伤害
4. 显示爆炸特效（冲击波 + 中心闪光 + 碎片粒子 + 烟雾 + 火花）
5. 伤害数字显示为红色暴击样式

**伤害公式**：`player.effectiveDamage × 8.0`
- 基础状态（攻击=5）：**40 伤害**
- +2 伤害加成（攻击=7）：**56 伤害**

## 四、副武器 — 手雷（Grenade）🔴

| 属性 | 值 |
|------|----|
| ID | `grenade` |
| 类型 | `aoe_delayed`（延时范围伤害） |
| 冷却时间 | 10 秒 |
| 伤害倍率 | 基础攻击 × 1500%（×15.0） |
| AOE 半径 | 150 px |
| 引信时间 | 5 秒（落地后延时爆炸） |
| 飞行时间 | 0.6–1.6 秒（随距离动态） |
| 特效颜色 | 红色 `#e74c3c` |
| 状态 | **已实装** |

**攻击流程**：
1. 自动搜索最近敌人位置
2. 从玩家向目标位置抛物飞行，带橙色火花尾迹
3. 落地后进入引信阶段：
   - 闪烁发光（黄→红渐变）
   - 爆炸圈范围提示
   - 头顶显示引信倒计时（秒数）
   - 持续喷射火花粒子
4. 5 秒后爆炸：大范围 AOE 伤害 + 多层冲击波 + 碎片 + 烟雾

**伤害公式**：`player.effectiveDamage × 15.0`
- 基础状态（攻击=5）：**75 伤害**
- +2 伤害加成（攻击=7）：**105 伤害**

- 基础状态（攻击=5）：**+100 HP**
- +2 伤害加成（攻击=7）：**+140 HP**

## 六、通用机制

### 伤害挂钩公式
所有副武器的数值都与玩家当前基础攻击力挂钩：

```
实际数值 = player.effectiveDamage × weapon.damageMult
```

其中 `effectiveDamage = PISTOL_DAMAGE(5) + damageBonus(来自能力升级)`

### 开发控制
开发者面板（按 **D 键**切换）可独立控制：
- 主武器攻击开启/关闭
- 1 号副武器开启/关闭
- 2 号副武器开启/关闭
- 3 号副武器开启/关闭
