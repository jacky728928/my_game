# 地图道具设计 - 墙体系统

## 核心设计

### 三层墙体

| 类型 | 颜色 | 阻挡移动 | 阻挡平射子弹 | 阻挡榴弹/手雷 |
|------|------|----------|--------------|---------------|
| **矮墙** | `#6b8e4e`（浅绿灰，灌木感） | ✓ | ✗ | ✗ |
| **中墙** | `#9a9a9a`（亮灰，混凝土） | ✓ | ✓ | ✗ |
| **高墙** | `#5a5a8e`（蓝紫灰，厚重障碍） | ✓ | ✓ | ✓ |

**天基屠龙炮**：无视所有墙体（空对地轰炸）

---

## 俯视外观

```
矮墙     ■■           小型，贴近地面颜色（灌木/掩体）
中墙     ████         标准方块，混凝土质感
高墙     ████████     最大，深色，厚重感
```

---

## 墙体布局

世界尺寸：2400 × 2400 px。生成于 `js/walls.js → generateWalls()`，在 `js/game_core.js → init()` 中调用。

| 区域 | 墙体类型 | 数量与位置 |
|------|---------|-----------|
| 世界中心 | 中墙 | 1 组十字形（横向 160px + 纵向 160px，各 40px 厚） |
| 四角 | 矮墙 | 4 组（每组 80×20 + 20×40，模拟灌木/掩体） |
| 边缘区域 | 中墙 + 矮墙 | 3 处（每处 80×40，按位置分配中/矮墙） |
| 外围圆环 | 高墙 | 8 块（等角分布于半径 900px 的圆环，限制玩家在中央活动区） |

---

## 碰撞规则

```
玩家 / 敌人移动 → 三种墙全部阻挡（由 resolveCircleWalls() 修正位置）

子弹碰撞：
- 矮墙子弹穿过
- 中墙子弹（直线）被阻挡
- 高墙子弹（直线）被阻挡

抛射物碰撞（榴弹/手雷飞行）：
- 矮墙/中墙穿过
- 高墙命中则提前爆炸

技能穿墙：
- 闪烁突袭：落点在墙内时自动向玩家方向回退到合法位置；路径不检测
- 冲刺翻滚：落点在墙内时同样执行回退修正
- 天基屠龙炮：无视所有墙体
```

## 实现参考

```javascript
// 墙体数据结构（常量定义位于 js/constants.js）
const WALL_TYPE = { LOW: 'low', MID: 'mid', HIGH: 'high' };
const WALL_COLORS = { low: '#6b8e4e', mid: '#9a9a9a', high: '#5a5a8e' };
const WALL_BLOCKS_DIRECT = { low: false, mid: true, high: true };
const WALL_BLOCKS_ARCING = { low: false, mid: false, high: true };

// 单块墙对象（运行时位于全局 walls[]）
const wall = { x, y, w, h, type: 'low' | 'mid' | 'high' };

// 核心 API（位于 js/walls.js）
function generateWalls();                                   // 生成布局，写入全局 walls[]
function resolveCircleWalls(oldX, oldY, radius, newX, newY); // 圆形移动碰撞修正（玩家/敌人）
function checkBulletWalls(prevX, prevY, newX, newY);         // 子弹直线采样碰撞，返回被阻挡墙体
function checkArcingProjectileWalls(x, y, radius);           // 抛射物圆形碰撞，仅高墙阻挡
```

### 调用方一览

| 系统 | 文件 | 调用的墙体 API |
|------|------|---------------|
| 玩家移动 | `js/player.js` | `resolveCircleWalls()` |
| 敌人追踪 | `js/enemy.js` | `resolveCircleWalls()` |
| 子弹 | `js/bullet.js` | `checkBulletWalls()` |
| 榴弹炮 | `js/combat.js` | `checkArcingProjectileWalls()` |
| 手雷 | `js/combat.js` | `checkArcingProjectileWalls()` |
| 闪烁突袭 | `js/active_skill.js` | `circleRectCollide()`（落点回退逻辑内联） |
| 冲刺翻滚 | `js/active_skill.js` | `circleRectCollide()`（落点回退逻辑内联） |
| 敌人生成 | `js/game_core.js` | 避开中墙/高墙生成 |
| 渲染 | `js/renderer.js` | 遍历 `walls[]` 绘制矩形 |
| 小地图 | `js/minimap.js` | 遍历 `walls[]` 绘制小矩形 |
