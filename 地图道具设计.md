# 地图道具设计 - 墙体系统

## 核心设计

### 三层墙体

| 类型 | 颜色 | 阻挡移动 | 阻挡平射子弹 | 阻挡榴弹/手雷 |
|------|------|----------|--------------|---------------|
| **矮墙** | `#5a7a5a`（暗绿） | ✓ | ✗ | ✗ |
| **中墙** | `#7a7a7a`（灰色） | ✓ | ✓ | ✗ |
| **高墙** | `#4a4a6a`（暗蓝灰） | ✓ | ✓ | ✓ |

---

## 俯视外观

```
矮墙     ■■           小型，贴近地面颜色（植被/灌木感）
中墙     ████         标准方块，灰色混凝土
高墙     ████████     最大，深色，厚重感
```

## 碰撞规则

```
玩家/敌人移动 → 三种墙全部阻挡

子弹碰撞：
- 矮墙子弹穿过
- 中墙子弹（直线）被阻挡
- 高墙所有子弹（直线+榴弹）被阻挡
```

## 实现参考

```javascript
// 墙体数据结构
const wall = {
  x, y,        // 左上角坐标
  width,       // 宽度
  height: 20,  // 固定高度（俯视角度只用宽高画矩形）
  type: 'low' | 'mid' | 'high',
};

// 子弹碰撞检测
function checkBulletWallCollision(bullet, walls) {
  for (let wall of walls) {
    if (wall.type === 'low') continue; // 矮墙不挡子弹
    
    if (wall.type === 'mid') {
      // 中墙挡平射
      if (lineIntersectsRect(bullet, wall)) return wall;
    }
    
    if (wall.type === 'high') {
      // 高墙挡所有
      if (lineIntersectsRect(bullet, wall)) return wall;
    }
  }
  return null;
}
```
