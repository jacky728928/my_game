# 生存射击 · 项目总览

## 概念
手机端俯视角生存射击。主角屏幕中央不动，虚拟摇杆移动世界，自动瞄准杀敌。

## 技术
HTML5 Canvas | 原生JS零依赖 | 手机触屏优先 | 兼容鼠标

## 文件地图
```
index.html              → 入口，加载顺序
css/style.css           → 全部样式
js/constants.js         → 数值常量、敌人预设、颜色 
js/player.js            → 玩家类 (HP/移动/攻击冷却)
js/enemy.js             → 敌人类 (6级预设/追踪/碰撞)
js/bullet.js            → 子弹类 (飞行/寿命)
js/input.js             → 虚拟摇杆 (触屏+鼠标)
js/minimap.js           → 小地图
js/renderer.js          → 主渲染 (背景/实体/血条/击杀/死亡)
js/main.js              → 主循环/生成/碰撞/状态机
```
