# 生存射击 · v0.2

手机俯视角生存 | Canvas 原生 JS | 摇杆操控

## 文件地图 & 改哪里
| 文件 | 内容 | 改什么 |
|------|------|--------|
| `index.html` | 入口/加载顺序 | 新增JS引用 |
| `css/style.css` | 样式 | 摇杆/布局 |
| `js/constants.js` | 所有常量 | 数值/敌人预设/颜色 |
| `js/player.js` | 玩家(HP/移动/攻击/等级) | 玩家逻辑 |
| `js/enemy.js` | 敌人(追踪/碰撞/6级) | 敌人AI |
| `js/bullet.js` | 子弹(飞行/碰撞) | 子弹逻辑 |
| `js/xp_orb.js` | 经验球(浮动/飞向吸收) | 经验球逻辑 |
| `js/input.js` | 虚拟摇杆 | 操控 |
| `js/minimap.js` | 左上小地图 | 小地图 |
| `js/renderer.js` | 主渲染(背景/实体/血条/UI) | 画面/UI |
| `js/main.js` | 主循环(生成/碰撞/状态机) | 游戏循环 |
